# AUR客户端

*必须在[archlinuxcn仓库](切换根目录（chroot）/6.6.补充软件和补充仓库.md#补充仓库)添加且密钥环安装后操作！*

## Paru 

安装`paru`:

    sudo pacman -S paru

paru的语法与pacman一致，同时他还把yay的go语言换成rust。

    paru -S <package_name> # 安装AUR和仓库包
    paru -R <package_name> # 卸载AUR和仓库包
    paru -Syu # AUR与仓库包统一更新

## yay

安装`yay`:

    sudo pacman -S yay

yay的语法与pacman一致，在这里我就不展开说了。
