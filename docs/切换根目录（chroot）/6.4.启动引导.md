# 启动引导

## 使用GRUB启动引导

安装GRUB：

    pacman -S grub efibootmgr

### *顺便插一嘴*

pacman是一个功能较为丰富的包管理器，主要是给Arch和Arch系发行版使用的。

常用语法：

    pacman -S <package_name> # 安装

    pacman -R <pacman_name> # 卸载

    pacman -Syu # 更新

其余用法详见[熟悉pacman](../7.熟悉pacman.md)

## 将GRUB引导装在/boot分区中

使用默认target(x86_64-efi)。命令如下：

    grub-install --efi-directory=/boot --bootloader-id=ARCH

修改`/etc/default/grub`文件，将`GRUB_CMDLINE_LINUX_DEFAULT`中把`loglevel`从3调到5。
