# Host相关

## hostname(主机名)

使用VIM编辑`/etc/hostname`,不要使用特殊字符和空格：

    xuhongdi-archpc

### *多说一嘴：VIM基本操作*

1.使用I键进入插入模式，esc键关闭输入模式。

2.默认模式下键入`:wq`即可保存并退出。

3.`:q`可以直接退出，`:q!`是删除缓冲区并退出。

4.`vim <file_name>`是可编辑文件的打开命令

编辑`/etc/hosts`:

    # Standard host addresses
    127.0.0.1  localhost
    ::1        localhost ip6-localhost ip6-loopback
    ff02::1    ip6-allnodes
    ff02::2    ip6-allrouters
    # This host address
    127.0.1.1  xuhongdi-archpc.localdomain xuhongdi-archpc
