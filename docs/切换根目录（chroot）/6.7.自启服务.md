# 自启服务

## 常规自启服务

| 自启服务 | 作用 |
| ------- | ---- |
| `networkmanager` | 网络工具 |
| `bluetooth` | 蓝牙 |
| `sddm/gdm/lightdm` | 显示管理器 |

## 安装自启服务

1.networkmanager

参考[pacstrap](../5.基本系统安装.md#pacstrap安装系统)中的命令。

若未安装：

    sudo pacman -S networkmanager

2.bluetooth

安装`bluez`：

    sudo pacman -S bluez bluez-utils

3.sddm/gdm/lightdm

参考[桌面环境](../8.桌面环境.md)，安装匹配的显示管理器。

## 激活自启服务

    sudo systemctl enable networkmanager # 网络管理器
    sudo systemctl enable bluetooth # 蓝牙
    sudo systemctl enable sddm # KDE显示管理器，也可以换成别的（lightdm/gdm）
