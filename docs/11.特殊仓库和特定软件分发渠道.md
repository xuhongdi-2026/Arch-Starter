# 特殊仓库和特定软件分发渠道

难度：6.0

## 特殊仓库

### testing 仓库

下面，让我们来讲解testing仓库。

testing仓库分为三种：

    [core-testing] # 核心测试仓库
    [extra-testing] # 扩展测试仓库
    [muitilib-testing] # lib32测试仓库

请注意，这些仓库比arch的标准仓库的更新快，一定要谨慎开启！

### unstable 仓库

unstable仓库包括：

    [gnome-unstable] # GNOME 不稳定版
    [kde-unstable] # KDE 不稳定版

他的目的是为了让极客尝试更新的桌面环境，不建议普通用户使用。
