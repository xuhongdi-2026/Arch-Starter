# 桌面环境

*本教程只裁定KDE Plasma，其余桌面环境自己配去！*

## 字体

桌面环境所需的字体其实是为了考虑英文环境的，但中文环境却没有适配。所以需要安装中文字体。

安装`noto-fonts-cjk`：

    sudo pacman -S noto-fonts-cjk

emoji字体：

    sudo pacman -S noto-fonts-emoji

## KDE Plasma

安装KDE Plasma:

    sudo pacman -S plasma-meta dolphin konsole

字体选择`noto-fonts`

安装成功后启动服务：

    sudo systemctl enable --now sddm # 6.6+则使用plasma-login-manager

输入用户密码后，进入桌面环境

*注：breeze默认sddm主题存在fallback的bug，建议切换到其他sddm主题后再切回来。*

在添加archlinuxcn仓库的情况下，安装pamac-aur

    sudo pacman -S archlinuxcn/pamac-aur

使用此包管理器即可。

[探索KDE](https://userbase.kde.org/Applications/Desktop/zh-cn)

## GNOME 

*这不是我主张推行的桌面环境*

安装`gnome`软件包组

    sudo pacman -S gnome

安装成功后启动服务

    sudo systemctl enable --now gdm

安装并启用[插件](https://extensions.gnome.org)

    kimpanel(fcitx5-GNOME美化插件)
    blur-my-shell(GNOME Shell模糊效果插件)
    App Menu
    Place Status Indicator
    Tray Icons(后台常驻)
    User Theme(GNOME Shell主题支持)
    # 可选
    Live Lock Screen

安装Tweak(优化)

    sudo pacman -S gnome-tweaks
