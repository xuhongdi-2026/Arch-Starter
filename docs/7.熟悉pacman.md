# 熟悉pacman

pacman是一个比较优秀的包管理工具，基于libalpm,提供丰富的包管理功能。

## 学习pacman

pacman基本语法

    sudo pacman -S <package_name> # 安装软件包
    sudo pacman -R <package_name> # 卸载软件包
    sudo pacman -Syu # 更新

pacman query（查询）语法

    pacman -Q <package_name> # 查询软件包
    pacman -Qi <package_name> # 了解软件包的更多细节
